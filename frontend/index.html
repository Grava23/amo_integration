Воронка и этапы, по которым ИИ-ассистент ведёт сделку. Используются эндпоинтами /api/v1/leads/resolve и /api/v1/leads/:id/transition.

amoCRM API токен

Статичный долгоживущий Bearer-токен для вызовов amoCRM (как в n8n). После сохранения не отображается.
